# 标准化：思考

## 概述

“遗忘是思考的第一公民”

## 概念化

命名：可溯源的创造性认知外化的迭代式 CODE 循环
目的：解构思维的神圣黑箱，把最私密的认知过程变成可观测、可审计、可复现的公共剧场。
方法：把人类早已在做的外化实践（笔记、草稿、对话、Git 历史）系统化、实时化、结构化。
预期效果：减少人记忆和推理负担、传播思考方法。

层次：
- 思考内容：类比议题
- 思考框架/结构：类比议程。认知工程学的术语为“情境意识”。
- 思考模式：类比议事规则。认知工程学的术语为“心智模型”。

流程：
1. C 澄清/收集：
2. O 组织/联想：
3. D 精炼：展示注意和遗忘过程，展示各种可能的思考路径。
4. E 表达

内容：
1. C 澄清：Notion ，核心是 Doubt
2. O 联想：意图，用来最什么。
3. D 遗忘/回忆/精炼：Thought 的 Insight。
4. E 表达：Note with 完整的 Thought

框架：
1. C 收集器
2. O 联想器：找多个想法的关联
3. D 精炼器：压缩和遗忘原始想法形成更精炼的想法。
4. E 表达器：Memo with 更系统的想法

## 信息化

### 接收选项

枚举值：接收、拒绝、悬疑

1. 接收信息

· 特征：当下可采纳、有共鸣、与目标相关
· 示例：可用观点、待办事项、可复用结论
· 处理：进入长期记忆

2. 否决信息

· 特征：不成立、不相关、已过时
· 示例：错误猜测、无效路径、临时内容
· 处理：直接丢弃

3. 悬疑信息

· 特征：暂时无法判断、需更多上下文
· 示例：待验证假设、不完整线索、潜在关联
· 处理：暂存缓冲区，附带悬疑标记


### 分享意图状态

布尔值，默认为 0。是否想要分享，是思考成熟的标志。
