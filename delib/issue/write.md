# 议题：写作

写作档案根据体裁分类，包括：
- 新闻写作：主要用于公众号等新媒体平台发布。
  - 札记：叙事新闻风格的工作日志。类似于创始人创业札记。交付仓库为“量潮科技工作札记 quanttide-essay-of-business-entity”。
  - 通知：硬新闻风格的各类正式通知。从公共关系档案移动到写作档案。交付仓库为“量潮科技工作通告 quanttide-notice-of-business-entity”
- 技术写作：主要用于各类技术文档。
   - 教程
   - 手册
- 学术写作：主要用于各类深度文章。
   - 报告
   - 论文
- 创意写作：用于各类虚构作品。 
   - 小说
