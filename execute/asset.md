# 执行：数字资产

## 第二大脑

为每个元仓库都建立一个 meta 文件夹，基本架构是 metadata 客体和 self主体，或者 合二为一的 index。
让第二大脑的“自我认识”显性化，方便人类和它对齐。
在有多级元仓库的时候，上层元仓库维护到下层元仓库的 meta，下层自己管理自己，边界更有限，避免过度。
index文件更新上述说明。
