# 学习笔记

本目录记录学习与成长过程中的知识积累。

## 笔记规范

### 命名规则

- 使用英文或中文命名
- 使用下划线 `_` 分隔词汇
- 示例：`llm.md`、`qtadmin_vs_qtcloud_think.md`

### 目录结构

```
learn/note/
├── index.md           # 内容摘要（必选）
├── README.md          # 目录说明（可选）
├── code/              # 代码相关
│   └── index.md
├── meta.md            # 元学习
├── connect.md         # 连接学习
├── llm.md             # LLM 学习
├── infra.md           # 基础设施
└── write.md           # 写作学习
```

### 编写原则

1. **原创优先**：记录个人理解和思考，而非简单复制
2. **来源标注**：引用外部内容时注明来源
3. **持续更新**：笔记是可迭代的，定期回顾和补充
4. **关联思考**：与其他笔记建立链接，形成知识网络

### 内容模板

```markdown
# 笔记标题

## 概述
一句话描述主题

## 核心内容
主要知识点

## 关联
- 关联笔记：[链接](./other.md)
```

## 更新流程

1. 创建新笔记 → 更新 index.md 主题列表
2. 创建新目录 → 创建 index.md 和 README.md
3. 重要变更 → 在 CHANGELOG.md 记录
