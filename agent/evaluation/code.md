“修改一轮以后即可达到类似分数”，这说明：大模型实际上储备了大量高阶知识，但在默认状态下倾向于输出“安全、通用、浅层”的内容。
这验证了一个重要的 Prompt Engineering 原则：如果你想要专家级的回答，你必须先用专家级的标准去审视它。

评分员的价值观直接影响结果。
如果你隐含的评判里就带着偏向性的提示，会不会导致大模型的收敛实际上是你引导的。
揭示了 Prompt Engineering 中一个关键风险：我们往往只能得到我们“想看到”的答案，而不是“正确”的答案。
高分不代表绝对正确，它只代表最符合评分员价值观。
