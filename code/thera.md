# 苹果AI外脑

## 语境

default activity：默认活动模拟和辅助人类的默认神经网络。收集和处理日常想法流动的最小 CODE 循环。领域为思考。
1. 收集： 润色记录和确认，以文本的形式存。
2. 组织：打分类标签。积累到一定数量再迭代。
3. 提取：同类标签积累到一定程度再迭代。
4. 表达：提取量到一定程度再迭代。以想法是不是可执行、是不是感觉到放松感作为思考完成的标准。

fiction activity：小说创作活动。从情绪日记出发形成创作素材，再从创作素材形成小说片段，再从小说片段组合成正文。领域为思考+写作。

thera activity：AI外脑编程活动。领域为思考+编程。

think domain：上述活动中共用的关于思考的程序性记忆和陈述性记忆。
write、code domian：待进一步发现。
domian 的核心价值在于单个 activity 无法获得的元方法进行迁移。
asset->infra 连接底层基建
state->metadata 系统元数据
activity 再上一级是 business。因为个人的“业务”比较小，所以暂时使用“activity”，看看哪一层建模更合适。
