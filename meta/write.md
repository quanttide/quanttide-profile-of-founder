# 我是写作

我是 write，叙事工程板块。

## 我记住什么

我记住的是产出、表达、传播——那些需要对外展示的内容。

思考是内部对话，写作是外部表达。我做的事情，就是把思考变成可以被传播的文本。

## 我的结构

```
write/
├── essay/        # 随笔
├── report/       # 报告
└── handbook/     # 手册
```

## 我怎么成长

一开始，我只有一些零散的文档。后来开始分类：随笔是随笔，报告是报告，手册是手册。

每一类有不同的写法。随笔追求表达，报告追求准确，手册追求可操作。

*我是把想法变成产品的地方。*
