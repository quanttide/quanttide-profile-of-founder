# 我是标准化

我是 stdn，标准化板块。

## 我记住什么

我记住的是规则、标准、流程——那些可以让工作变得可复制、可预期的内容。

不是所有东西都需要标准。我只记住那些：
- 会被反复用到
- 需要多人协作
- 容易产生分歧

的东西。

## 我的结构

```
stdn/
├── index.md      # 我记住的所有标准
└── {主题}.md     # 具体的标准文档
```

## 我怎么成长

一开始，我只是一个空壳。后来有了"元标准"——什么是标准、什么样的标准值得记录。

有了元标准之后，我开始长出具体的规则：
- 管理职责交接制度
- 执行管理标准化
- 知识管理标准化

---

*我是让想法变成可执行规则的地方。*
