# Meta 目录

元数据层——量潮创始人档案的自我认知中心。

## 目录结构

```
meta/
├── README.md     # 本文件：目录说明
├── metadata.md   # 版本档案：档案的版本信息与发布记录
└── self.md       # 自我认识：档案的身份认同与能力边界
```

## 设计理念

Meta 目录体现了"元认知"的思想——档案的自我认知本身也是可版本化、可追溯的知识。

- **README**：档案的"操作手册"，告诉你如何理解这个目录
- **metadata**：档案的"档案卡"，记录我从哪里来、现在在哪
- **self**：档案的"自我认识"，我如何看待自己
