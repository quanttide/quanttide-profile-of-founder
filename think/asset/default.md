# 思考：数字资产

## 想法1: 公开档案

创始人档案这段时间的收集已经进入了良好的状态。在这里完成了第一轮分类和澄清。接下来把一部分放到公司档案移交，一部分移到领域档案继续澄清。这是一个非常理想的批量处理海量碎片化信息的工作流。

主要难点在于把隐式的维护规则逐渐显式化，并且围绕这些显式的规则研发平台。

优化关于工作流的工作流，是“元工作”。
信息处理从手工业到工业化。

产能在进一步提高，工作热情被打开，又比较害怕面对自己制造的混乱。
