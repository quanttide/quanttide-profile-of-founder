# 思考

思考是认知工程的核心产出，记录创始人对各个领域的深度思考。

## 主题

- **认知与智能**：智能体的设计、元认知工程、AI 外脑
- **产品与商业**：产品思维、商业模式、资产运营
- **组织与管理**：DevOps、HR、IAM、组织架构
- **个人成长**：自我认知、健康、课程学习
- **知识与写作**：知识管理、写作方法

## 文档结构

每个主题文件（如 `agent.md`、`product.md`）采用统一的章节结构：

| 章节 | 说明 |
|------|------|
| `## 收集` | 记录想法、与 AI 对话的整理 |
| `## 组织` | 对收集内容进行组织和分类 |
| `## 提取` | 从组织内容中提取核心洞察 |
| `## 表达` | 整理为最终可发布的成果 |

## 内容规范

### 来源

- 与 AI 对话的深度总结
- 个人想法和复盘
- 实践经验的提炼

### 子主题标记

- `### 想法：` 个人想法或问题
- `### 复盘：` 对某个方法的复盘总结

### 格式

- 标题使用 ATX 风格（`#`、`##`、`###`）
- 列表使用 `-` 或 `1.`
- 引用块使用 `>` 标注重要结论
