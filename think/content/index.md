# 思考

思考是认知工程的核心产出，记录创始人对各个领域的深度思考。

## 目录结构

```
think/
├── content/     # 主题笔记（原有文件迁移）
├── framework/  # 思考框架（待建设）
└── pattern/    # 思考模式（待建设）
```

## 主题

- **认知与智能**：智能体的设计、元认知工程、AI 外脑
- **产品与商业**：产品思维、商业模式、资产运营
- **组织与管理**：DevOps、HR、IAM、组织架构
- **个人成长**：自我认知、健康、课程学习
- **知识与写作**：知识管理、写作方法

## 关联

- 智能体 → 思考 → 智能体工程化
- 产品 → 思考 → 产品标准
- 写作 → 思考 → 写作标准
