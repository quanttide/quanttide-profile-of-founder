# 智能体安全审计

“也借这个机会再推一下agent auditing（审计） 这个方向，包括我们刚发布的工具：https://github.com/HeadyZhang/agent-audit

最近我们使用 agent-audit 对 ClawHub 上全部 18,899 个 OpenClaw Skills 进行了扫描，这是目前规模最大的 OpenClaw 生态安全审计之一。

一些核心发现：

1. 共检测到 13,355 个安全问题，其中 1,492 个为最高严重级别（BLOCK）

2. 859 个 skill 存在严重漏洞（占比 4.5%）

3. 带 scripts/ 目录的 skill 漏洞率 46.6%，不带的 16.8%，风险差 2.8 倍

4. 发现真实恶意软件攻击：多个账号发布包含 base64 编码后门 的 skill，解码后会连接 C2 服务器 下载第二阶段载荷

5. 检测到 641 个混淆 shell 命令、457 个身份文件篡改尝试、250 个虚假依赖社工攻击

可以预见，随着 agent 系统越来越复杂，安全审计、自动检测、以及防护机制都会变得越来越重要。未来我们也会继续在 agent 安全、审计与防护 方向推进研究，并在论文之外发布更多真正可用的开源工具。”
